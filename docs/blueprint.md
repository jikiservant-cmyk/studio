# **App Name**: Beacon

## Core Features:

- Sermon Streaming: Live audio and video streaming of sermons and services.
- Event Calendar: Display upcoming church events, services, and meetings with registration option.
- AI Scripture Finder: An AI powered tool that answers user's questions using relevant Bible verses.
- Donation Portal: A secure and user-friendly donation platform with recurring donation options.
- Contact Form: Contact form for inquiries and prayer requests.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and stability.
- Background color: Light gray (#F5F5F5) to ensure readability and a sense of calm.
- Accent color: Soft gold (#FFD700) for highlights and important calls to action.
- Body font: 'PT Sans', a humanist sans-serif, for readability.
- Headline font: 'Alegreya', a humanist serif, for an elegant and intellectual feel.
- Use clear, minimalist icons representing church activities and values.
- Subtle fade-in effects and smooth transitions on page load and navigation.
